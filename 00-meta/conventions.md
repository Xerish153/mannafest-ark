# Vault Conventions

## Internal References
- Use wikilinks for all internal references: `[[note-name]]`
- Organize cross-references using backlinks

## File Naming
- All filenames use kebab-case: `my-note-title.md`
- No spaces or underscores in filenames
- Prefix with numbers for chronological ordering where applicable

## Frontmatter
- No auto-frontmatter in notes
- Metadata managed through wikilinks and folder structure

## Session Handoffs
- Format: `YYYY-MM-DD-handoff-N.md`
- Store in 03-sessions/ folder
- Example: `2026-04-16-handoff-1.md`
