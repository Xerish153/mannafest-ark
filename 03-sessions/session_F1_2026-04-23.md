## Decisions and deviations

- **Shared primitives are wrappers, not rebuilds.** Three of four F1 "shared primitives" already existed in Batch 3's diagram library under different names. Rather than duplicate, built thin wrappers at `src/components/featured/primitives/` that re-export under the F1 canonical names and add a `variant` prop for page-specific styling hooks. `DualTreeVisual` is the only fresh build. This matches Batch 3's session-record pattern of preserving existing code when it already meets the spec.
- **Scarlet Thread hero follows the brief, not the prompt** (per [[F1_addendum_01]] Item 4). Bento grid with SVG ribbon overlay and 2×-weight Yoma 39b tile — not the prompt's "eight-thread vertical visual." `<ScarletRibbonBento />` is bespoke; it does not consume `ThreadTimeline`.
- **Seed Promise depth-1 renders 4 inflection points, not the brief's 12 stops.** F1 prompt set the count; the brief is richer. A post-F3 hygiene pass can expand to the 12-stop ribbon. Flagged in the drilldown body_markdown for future reference.
- **Suffering Servant manuscript hero uses an SVG scroll-texture placeholder, not licensed manuscript photography.** Per the Suffering Servant brief's sourcing caveat: cite text and metadata, link to the Israel Museum Digital Dead Sea Scrolls archive rather than reproducing plates on-page.
- **Drilldown commentary uses inline `<CommentarySection />` for the primary verse only.** Per-verse Doctrine A commentary for every verse in a drilldown's passage (e.g. all 15 verses of Isa 52:13–53:12) would require Pastor Marc's super-admin curation and is Batch 18's scope. F1 ships the scaffolding; deeper curation lands in the Isaiah §7.7 retrofit.
- **`/study/* ` deletion pattern** follows [[project_mannafest_study_route_retirement]] memory — delete fresh, rebuild fresh, never lift-and-shift. Inbound-link grep caught 6 references; all updated.
